# تحليل توزيع السكان باستخدام Power BI | Population Distribution Analysis

هذا المشروع يعرض توزيع السكان حسب الولايات الأمريكية باستخدام Power BI، ويشمل خرائط تفاعلية وتحليل حسب الجنس.

This project visualizes population distribution across U.S. states using Power BI, including map visuals and gender-based breakdowns.
