# تحليل توزيع السكان باستخدام Tableau | Population Distribution Analysis

هذا المشروع يعرض توزيع السكان حسب الولايات الأمريكية باستخدام Tableau، ويشمل خرائط تفاعلية وتحليل حسب الجنس.

This project visualizes population distribution across U.S. states using Tableau including map visuals and gender-based breakdowns.
